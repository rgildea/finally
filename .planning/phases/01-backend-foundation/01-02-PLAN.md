<objective>
Create the FastAPI application layer that wires the completed market data subsystem and the Wave 1 database layer together: a lifespan context manager that initializes the database and launches the market data polling loop on startup (APP-01), a conditional static-file mount that serves the Next.js export when present (APP-02), and a `/api/health` endpoint (APP-03). Getting the lifespan orchestration and clean shutdown right is what makes `uvicorn app.main:app` start and stop without errors.

| client → /api/health | Public unauthenticated probe endpoint; no inputs accepted |

| env → app config | load_dotenv() reads operator-controlled .env (MASSIVE_API_KEY, DB_PATH); not user-supplied |
| process → market source / DB | Lifespan owns the background polling task and DB init; no external input reaches them in this phase |
